# Product Overview

## Feature Flag Rules Engine

A minimal feature flag evaluator designed to demonstrate spec-driven development using Kiro IDE.

### Core Functionality
- **Input**: User context (userId, region, plan)
- **Output**: List of enabled features for the user
- **Rules**: Static rule evaluation (e.g., Pro users get Feature X)

### Development Philosophy
- **Spec-driven development**: All features must be specified before implementation
- **Clear requirements**: Ensure testable behavior through upfront specification
- **Minimal viable implementation**: Focus on core functionality first
